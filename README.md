# Telegram Music Downloader v6

Программа для скачивания аудио и музыки из Telegram-групп и каналов.

## Особенности
- Авторизация через API ID и API Hash
- Поддержка потокового сканирования и скачивания аудио
- Сохранение отчётов о скачанных файлах
- Настройки через `config.json`

## Установка
```bash
pip install -r requirements.txt


Использование:

Заполните auth.txt (API_ID, API_HASH, PHONE_NUMBER)

Запустите app_v6.py

Авторизуйтесь и выберите чат для скачивания
